## What is 67 Speed Test?

67 Speed Test is a free browser-based arm speed challenge game that uses your device's camera and Google MediaPipe Pose technology to track your arm movements in real-time. Move your arms as fast as you can for 20 seconds and see how you rank against millions of players worldwide.

**No downloads. No installs. Just your camera and 20 seconds.**

> 🎮 [Play 67 Speed Test Free Online](https://67-speed-test.github.io)

## Tech Stack

Built with modern browser technologies — no frameworks, no build tools, no server required.

- **Google MediaPipe Pose** — Real-time body tracking with 33 body landmarks
- **WebGL / WebAssembly** — On-device ML inference
- **Privacy-First** — All camera processing happens locally, zero data uploaded
